Projektspezifisches Framework anlegen (Oxygen)

Sie müssen ein projektspezifisches Framework als eine Erweiterung des ediarum.basis-Frameworks anlegen, auch wenn Sie vorerst keine Veränderungen an ediarum.BASE vornehmen möchten. Sie müssen im Arbeitsordner einen Projektordner anlegen und Oxygen mitteilen, dass alle Dateien Ihres projektspezifischen Frameworks dort liegen.

  1. In den Optionen von oXygen XML Author: Dokumenttypen-Zuordnung > ediarum.basis > Erweitern (erstellt eine Art Duplikat basierend auf Basisframework)
  2. > Speicherort "Extern": in Verzeichnis "Programme/Oxygen/frameworks/ediarum_[projektname]", unter dem Dateinamen "ediarum_[projektname].framework" speichern
