Neues Ticket anlegen

(1) Neue Tickets können über die Schaltfläche "Neues Ticket angelegt werden. In der Kategorie "Thema" sollte eine aussagekräftige Überschrift notiert werden. Es ist zwar bei Redmine nicht obligatorisch eine Beschreibung anzulegen. In der Beschreibung sollten auf jeden Fall Details zur Aufgabe festgehalten werden. Das ermöglicht zum einen Mitarbeiter_innen die Einsicht, zum muss zu einem späteren Zeitpunkt nicht entschlüsselt werden, was das eigentliche Problem war.

(2) Ein Ticket kann einzelnen Mitarbeiter_innen zugewiesen werden. Außerdem lässt sich das Ticket spezifizieren, indem es einer Kategorie zugeordnet wird. Idealerweise werden diese beiden Felder berücksichtigt, wenn ein Ticket angelegt wird. Mit der Kategorie wird also das Aufgabenfeld näher bestimmt. Jedes Projekt hat ein eigenes Aufgabenprofil.
