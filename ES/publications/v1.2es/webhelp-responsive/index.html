Instalación y configuración básica
La configuración básica de ediarum hace disponible un entorno de trabajo para ediciones filológicas. Después de una instalación y configuración existosa, los editores y editoras pueden crear datos XML conforme a TEI con registros, notas etc. en un entorno visual que es similar al de Word.
Sobre esta guía
La configuración básica primero hace disponible ediarum en la versión estándar, sin ninguna configuración específica de su proyecto editorial y sin que necesite conocimientos de programación.
Resumen de la configuración básica
ediarum está disponible en GitHub, en forma de componentes de configuración para Oxygen XML Author (los llamados frameworks) y para la base de datos XML eXist-db (las llamadas aplicaciones).
Tips para la instalación de eXist-db y Oxygen
Las guías de instalación para eXist-db y Oxygen se encuentran en sus sitios web correspondientes. Los consejos adicionales que se han recopilado aquí se basan en la experiencia de los desarrolladores y desarrolladoras de ediarum y pueden ser útiles para los problemas típicos de instalación.
Instalar y configurar la base de datos
Instala la aplicación eXist ediarum.DB. Usando esta aplicación, crea su propio proyecto en su base de datos que contiene todos las configuraciones necesarias para la interacción entre la base de datos y Oxygen XML Author.
Configurar el entorno de trabajo en Oxygen XML Author
Conecta Oxygen XML Author con eXist-db (la base de datos) y hace disponible todas las funcionalidades contenidas en ediarum.
Configuración específica de ediciones
Si su edicion requiere una definición de esquema XML más completo que el que cubre ediarum, puede extender el documento esquema y el entorno de trabajo. Además, puede crear registros adicionales.
Sobre los frameworks de Oxygen y las características específicas de ediarum
El proceso básico de configuración a través de los llamados “Frameworks” se encuentra en la documentación de Oxygen. Ediarum proporciona algunos módulos estándares para esta configuración.
Crear registros adicionales
Si necesita un registro que no está incluido por defecto en ediarum, puede crearlo en eXist-db en la aplicación ediarum.DB y luego vincularlo a Oxygen.
Integrar los fondos bibliográficos de Zotero como registro
Si gestiona sus fondos bibliográficos en Zotero, también puede integrarlos como registro en ediarum y así hacer referencias bibliográficas.
